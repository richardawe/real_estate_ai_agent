Setup required

GitHub PAT not configured

The site needs a Fine-Grained GitHub Personal Access Token to post workflow requests. Follow these steps once, then the site will work without any further sign-in.

  1. Go to github.com/settings/tokens → Generate new token
  2. Under Repository access, choose Only select repositories → real_estate_ai_agent
  3. Under Repository permissions, set:
    • Issues → Read and write
    • Contents → Read and write
  4. Click Generate token and copy the value
  5. Open site/config.js in the repository and replace YOUR_FINE_GRAINED_PAT with your token, then push to main

The token is stored in a public file — its scope is intentionally limited to this one repo (worst case: someone posts a comment on one of your issues).
